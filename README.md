# 🍔 Burguer Menu

## 📖 Sobre o Projeto

O **Burguer Menu** é uma aplicação web desenvolvida para praticar conceitos fundamentais e modernos do JavaScript, criando uma interface dinâmica para exibição e manipulação de itens de um cardápio de hamburgueria.

O projeto utiliza métodos avançados de manipulação de arrays para renderizar, filtrar e calcular informações dos produtos de forma dinâmica diretamente na interface.

---

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

---

## ⚙️ Especificações Técnicas

Este projeto foi desenvolvido utilizando:

- JavaScript `filter()`
- JavaScript `map()`
- JavaScript `reduce()`
- Template Strings
- Manipulação da DOM
- Functions
- Arrays e Objetos
- Eventos com JavaScript
- Renderização dinâmica de elementos

---

## 🎯 Objetivo do Projeto

O principal objetivo deste projeto foi aprofundar conhecimentos em JavaScript moderno, especialmente em:

- Manipulação de arrays
- Criação dinâmica de elementos
- Atualização de interface via DOM
- Organização de lógica com funções
- Interatividade no front-end

---

## 📸 Funcionalidades

✅ Listagem dinâmica de produtos  
✅ Filtragem de itens  
✅ Cálculo automático de valores  
✅ Interface interativa  
✅ Manipulação completa da DOM
